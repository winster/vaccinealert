# Alert for vaccine availability

This application is intended to support people who otherwise misses vaccination schedule which are available only for a short duration.

To use this application, all you have to do is to choose your district and subscribe. This application will send you web notifications
when a center in your district has open schedule for appointment.

### Collaborate

This application is a monolith, not designed to be data durable or scalable. If you think, it is worth to contribute, you are welcome
